# 摸鱼日常 -- 又名今日上朝 👑

> 皇上，今日您拉屎了吗？

一个超级可爱搞笑的日常活动追踪网页。记录你每天干了啥沙雕事，看看自己到底是"今日圣人"还是"超级废物"。


## 这是什么？

工作之余用来放（mo)松(yu)的搞笑小工具。每天打开"上朝"，勾选你今天完成的沙雕事项，系统会根据你的完成情况给出搞笑评级、颁发废物等级证书、解锁各种离谱成就。

**纯前端，无需后端，所有数据保存在浏览器本地。感谢兢兢业业的claude code。**

## 技术栈

- **HTML** — 单文件，开箱即用
- **Tailwind CSS** (CDN) — 样式
- **原生 JavaScript** — 零依赖
- **LocalStorage** — 数据持久化

## 数据说明

所有数据保存在浏览器 `localStorage` 中：

| Key | 内容 |
|-----|------|
| `jrsc_data` | 每日活动完成记录（按日期索引） |
| `jrsc_profile` | 已解锁成就、抚摸次数等 |

清除浏览器数据会导致记录丢失。如需备份，可在浏览器控制台执行：

```javascript
// 导出
copy(JSON.stringify({data: localStorage.getItem('jrsc_data'), profile: localStorage.getItem('jrsc_profile')}))

// 导入（粘贴你导出的 JSON）
const backup = JSON.parse('你的备份JSON');
localStorage.setItem('jrsc_data', backup.data);
localStorage.setItem('jrsc_profile', backup.profile);
location.reload();
```
